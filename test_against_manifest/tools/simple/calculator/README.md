# Calculator Tool

Simple arithmetic calculator tool for testing.

## Operations

- `add`: Addition
- `subtract`: Subtraction
- `multiply`: Multiplication
- `divide`: Division
- `health_check`: Health status

## Usage

```bash
python3 scripts/calculator.py '{"operation": "add", "a": 5, "b": 3}'
```

## Testing

```bash
python3 tests/test_calculator.py
```
